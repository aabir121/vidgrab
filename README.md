# VidGrab – Video Link Downloader

## Introduction
This extension, named "VidGrab – Video Link Downloader", extracts downloadable video links from webpages, lets users filter and select them, and saves downloads to a chosen subfolder of the browser's Downloads directory.

## Features
- Extracts video links from the active tab.
- Provides an interface in the popup to view and download extracted videos.
- Utilizes `downloads` permission for direct downloading.
- Works across all URLs (`<all_urls>`).

## Installation
1. Download or clone this repository.
2. Open Chrome and navigate to `chrome://extensions`.
3. Enable "Developer mode" by toggling the switch in the top right corner.
4. Click on "Load unpacked" and select the directory where you downloaded this extension.

## Usage
1. Navigate to a webpage containing a video.
2. Click on the extension icon in your browser toolbar.
3. Instructions on how to use the popup and download videos will be provided here.
