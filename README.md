# VidGrab – Video Link Downloader

## Introduction
This extension, named "VidGrab – Video Link Downloader", extracts downloadable video links from webpages and initiates direct downloads to the browser's default Downloads folder (no save dialog).

## Updated Features
- Extracts video links from the active tab and displays them in a user-friendly interface.
- Supports batch downloading of multiple video links at once.
- Downloads are saved automatically to the browser's default Downloads folder; users are not prompted for a save location.
- Utilizes `downloads` permission for direct downloading.
- Works across all URLs (`<all_urls>`).

## Installation
1. Download or clone this repository.
2. Open Chrome and navigate to `chrome://extensions`.
3. Enable "Developer mode" by toggling the switch in the top right corner.
4. Click on "Load unpacked" and select the directory where you downloaded this extension.

## Updated Usage
1. Navigate to a webpage containing a video.
2. Click on the extension icon in your browser toolbar.
3. Select the desired video links from the popup interface.
4. Click the download button to save the videos to your browser's default Downloads folder (no save dialog).
